# Local Cortex

Local Cortex is a local-first chat UI with a built-in editor, built with Tauri, React, and Vite.

## Requirements

- Node.js 18+
- Rust toolchain (stable)
- Tauri CLI (installed via npm or cargo)

## Setup

```bash
npm install
```

## Run (Web UI)

```bash
npm run dev
```

## Run (Desktop App)

```bash
npm run tauri dev
```

## Build

```bash
npm run build
npm run tauri build
```
